# CFN Roadmap

## Version 1.0

- Navigator for GLP-1 receptor agonists
- Studio for editing one drug class
- Save Studio JSON through `saveStudioContent()`
- Persist through Google Spreadsheet
- Preview saved content in Navigator

## Later

- Additional drug classes
- Stronger typed data model
- Separate build/deploy flow for `src/` and `html/`
- Tests for Service and Repository transformations

## Not In Version 1.0

- AI generation
- PubMed integration
- RNC Evidence Library integration
- Notion integration
- PMDA integration
- External APIs
