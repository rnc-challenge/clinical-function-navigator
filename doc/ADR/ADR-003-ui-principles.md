# ADR-003: UI Principles

## Status

Accepted

## Decision

Navigator is a patient support workspace.

Studio is a CMS for patient support content.

Neither view should reproduce the spreadsheet appearance.

## Consequences

- The UI is organized around Summary, Assessment, Education, Safety Management, Multidisciplinary Support, Red Flag, Functional Impact, Clinical Pearl, Evidence, and Guideline.
- Spreadsheet columns may change without requiring the Studio UI to change.
