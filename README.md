# Clinical Function Navigator (CFN)

> **患者支援に必要な視点を、一つに。**

Clinical Function Navigator（CFN）は、薬剤・疾患・生活機能・患者背景・エビデンスを統合し、患者支援に必要な情報へ迷わずたどり着くための臨床支援プラットフォームです。

---

## コンセプト

医療には多くの情報があります。

しかし、実際の現場では、

* 必要な情報が分散している
* 必要な確認事項が抜けてしまう
* エビデンスを探す時間がない
* SOAPや薬歴作成に時間がかかる

という課題があります。

CFNは、

**「情報を管理する」のではなく、患者支援につながる知識を構造化すること**

を目的として開発しています。

---

## CFNでできること

### 💊 薬効群から考える

薬効群ごとに、

* 確認すること
* 説明すること
* 支援すること
* Red Flag
* 生活機能への影響
* エビデンス
* ガイドライン

を一覧で確認できます。

---

### 🩺 疾患から考える

疾患に関連する薬効群や患者支援を横断的に確認できます。

---

### 🚶 生活機能から考える

食欲低下、転倒、排尿、睡眠、栄養状態など、生活機能を起点とした患者支援を行えます。

---

### 👤 患者背景から考える

高齢、CKD、妊娠、認知症など、患者背景に応じた注意点を確認できます。

---

## 患者支援ワークフロー

どの入口から開始しても、最終的には同じ流れになります。

```text
確認する
    ↓
説明する
    ↓
支援する
    ↓
記録する
```

CFNは、この一連の流れを支援するために設計されています。

## 開発状況

現在は Version 0.1 を開発中です。

最初の対象薬効群は **GLP-1受容体作動薬** とし、Clinical View・SOAP支援・エビデンス表示の実装を進めています。

---

## 開発方針

新しい機能を追加する前に、必ず次の問いを確認します。

> **この機能は、患者支援をより良くするか。**

Yesなら実装する。

Noなら実装しない。
